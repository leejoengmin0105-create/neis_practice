📋 학교자율시간 등록 절차

STEP 1 — 상단 조회 조건에서 학년도, 학교급, 교육과정을 선택하고 [조회] 버튼 클릭
STEP 2 — 우측 패널 [등록] 버튼 클릭 → 학교자율시간 등록 팝업 오픈
STEP 3 — 팝업에서 다음 항목 입력:
  • 운영방식 : 학급공통 / 학생선택
  • 편제(관련교과(군)) : 국어, 수학, 사회/도덕 등
  • 과목/활동 구분 : 과목 또는 활동
  • 과목 : [찾기] 버튼 → 과목찾기 팝업에서 검색 후 선택
  • 학기 : 1학기 / 2학기 선택
  • 시수 : 숫자 입력 (3~4학년 최대 29시간, 5~6학년 최대 32시간)
STEP 4 — 팝업 하단 [저장] 클릭 → 우측 목록에 추가 확인
STEP 5 — 등록 완료 후 화면 하단 [저장] 버튼으로 최종 저장
⚠ 유의사항
• 3~4학년: 학기당 교과 시수 최대 29시간 제한
• 5~6학년: 학기당 교과 시수 최대 32시간 제한
• 과목 선택 시 고시 내 과목은 검색되지 않으며, 고시 외 과목만 등록 가능
• 동일 과목은 한 학기만 편성 가능
• 적용 학년: 3~6학년 (1~2학년은 학교자율시간 미편성)
